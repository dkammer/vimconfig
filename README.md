# Personal VIM Configuration

Download via terminal: `curl -L -O https://github.com/dkammer/vimconfig/archive/refs/heads/main.zip`

Unzip in home directory: `unzip main.zip; mv vimconfig-main/.* .;rm -rf vimconfig-main`
